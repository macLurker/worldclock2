new WorldClock($span, details, tzopt, tfopt, dsopt, bsopt)

Parameters:
Name Type Attributes Default Description
$span jQuerySingleSpan a jQuery object representing the HTML `span` element to be transformed into a clock. Note that all existing content within the span will be will be removed.
details dictionary details of new clock display. See below.
tz string <optional>
 Europe/London The timezone for the clock.
tf number <optional>
 24 Time format: 12 for 12-hour clock or 24 for 24-hour clock
ds string <optional>
 false Whether clock displays seconds or not. "True" = display seconds
bs string <optional>
 false Whether hours/minutes/seconds separator blinks. "True" = cause the separator to blink
